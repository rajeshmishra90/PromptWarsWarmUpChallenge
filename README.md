# PromptWars Warm Up Challenge

This repository contains the solution for the warm up hackathon challenge: **A Cooking To-Do List**.

## Challenge Description
Build a simple AI micro-app that helps a user generate a personal cooking to-do list based on their day.
It should provide a structured meal planning flow that produces:
- Breakfast/Lunch/Dinner plan
- Grocery list
- Substitutions
- Budget feasibility logic
